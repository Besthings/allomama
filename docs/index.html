License Validator API

This is a simple API for validating license keys. It works by checking against a predefined list of valid keys.

Endpoint

GET /api/validate.html?key={license_key}&hwid={hardware_id}

Parameters

  • key: The license key to validate
  • hwid: The hardware ID of the machine

Response Format

{
    "success": true|false,
    "message": "Status message",
    "data": {
        "key": "license_key",
        "hwid": "hardware_id",
        "activated_at": "ISO timestamp",
        "last_checked": "ISO timestamp"
    }
}

How It Works

This API is a static JSON file hosted on GitHub Pages. It simulates a real API by using JavaScript to process the query parameters and return appropriate responses.

Important Note

This is a client-side implementation and should only be used for demonstration purposes or non-critical applications. For production use, consider implementing a proper server-side validation system.
